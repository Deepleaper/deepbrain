## Key Features

- 🧠 **Hybrid Search**: Semantic (vector) + keyword search with re-ranking
- 💬 **RAG Chat**: Multi-turn chat with inline citations, conversation memory summary
- 📊 **Knowledge Graph**: Auto-extracted entity relationships
- 🎴 **Flashcards**: Spaced repetition (SM-2) for active recall
- 🔄 **Dream Cycle**: Automated knowledge maintenance
- 📥 **Multi-source Import**: GitHub repos, YouTube transcripts, RSS feeds, Notion, Obsidian
- 🌐 **Web UI + MCP Server**: Browser playground and AI editor integration
- 🏗️ **Local-first**: PGLite (embedded Postgres) — no external database needed
- 🌍 **Multi-provider**: Ollama, OpenAI, Gemini, DeepSeek, Dashscope (Qwen), Zhipu (GLM), Moonshot (Kimi)
- 📦 **npm**: `npm install -g deepbrain`

## Tech Stack

- TypeScript, Node.js
- PGLite (embedded PostgreSQL with pgvector)
- agentkits (LLM abstraction)
- Apache-2.0 License
